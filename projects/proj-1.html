Autonomous Wheel-Loader Control Stack

Komatsu x CERLAB

CERLAB, Carnegie Mellon University | Prof. Kenji Shimada

Introduction

Efficient and safe operation of heavy construction machinery is a growing challenge in modern worksites. Autonomous wheel loaders can reduce operator fatigue, prevent accidents, and improve productivity in hazardous environments such as mining, quarrying, and large-scale construction. This project focuses on building a modular autonomy stack for Komatsu wheel loader that can perform loading and dumping tasks safely and reliably in simulation. The objective is to design, simulate, and control full-cycle autonomous behavior using advanced motion-planning and nonlinear control techniques.

Methodology

The autonomy stack was developed inside a ROS2 + Isaac Sim ecosystem with a modular architecture supporting perception, planning, and control. My work centered on the trajectory-planning and control subsystems:

Trajectory Planning:

  • Designed kinematically feasible trajectories for the articulated wheel loader using RRT, RRT*, and Hybrid A* algorithms.
  • Integrated vehicle-specific constraints such as articulation steering limits, minimum turning radius, and loader geometry.
  • Applied B-spline smoothing and velocity profiling to convert raw graph-search paths into dynamically feasible motion references.

Closed-Loop Control:

  • Implemented a nonlinear model predictive controller (NMPC) using the ACADO toolkit to track smoothed trajectories with real-time feedback.
  • Tuned system-level constraints (actuator limits, steering rate, velocity bounds) for stable control under varying task conditions.

Simulation Infrastructure:

  • Deployed the NMPC controller inside ROS2 and connected it to Isaac Sim using a dual-Docker architecture, enabling real-time closed-loop simulation.

Implementation

All modules were integrated as separate ROS2 nodes for clarity, testing, and reproducibility. The planner, smoother, and NMPC controller were benchmarked across multiple loading-cycle scenarios, including bucket approach, scooping, reversing, and material dumping.

Results

The integrated pipeline successfully generated collision-free, kinematically feasible trajectories and executed them stably inside Isaac Sim. The NMPC consistently maintained low cross-track error and smooth articulation angle transitions. The modular design now supports further expansions such as perception integration and reinforcement-learning-based policy layers.
