Autonomous Vehicle Control System

Tesla Model 3 Simulation

Introduction

Autonomous driving systems play a critical role in improving road safety, reducing traffic fatalities, and enabling efficient mobility in urban environments. This project focused on developing and evaluating a full-stack autonomous vehicle control pipeline for a Tesla Model 3 inside the Webots simulator. The objective was to design robust planning, control, and localization modules capable of handling real-time navigation tasks, even in GPS-denied environments and under noisy sensor conditions.

Methodology

The system was implemented as a modular stack consisting of vehicle modeling, trajectory tracking controllers, path planning, and localization:

Vehicle Dynamics Modeling

  • Constructed a linearized bicycle model of the Tesla Model 3 to capture essential kinematic behavior.
  • Tuned model parameters to match turning radius, wheelbase, and velocity ranges.

Control System Design

  • Implemented PID controllers for lateral (steering) and longitudinal (speed) tracking, enabling real-time response to reference trajectories.
  • Designed state-space controllers using pole placement to improve stability during aggressive steering or curved-road scenarios.
  • Developed a discrete-time LQR controller for optimal lateral control, balancing steering smoothness, noise rejection, and safety constraints.

Planning & Navigation

  • Programmed an A* planner to generate obstacle-aware, collision-free trajectories in dynamic environments.
  • Ensured safe navigation by incorporating road geometry, constraints, and obstacle boundaries.

Localization & Estimation

  • Deployed EKF-SLAM to estimate vehicle position and heading when GPS is unreliable or unavailable.
  • Modeled sensor noise and incorporated range/bearing measurements to improve state estimation quality.

Implementation

Each subsystem—vehicle model, controllers, planner, and SLAM—was implemented as a standalone module inside Webots. Controllers operated at discrete time steps to maintain real-time behavior, and SLAM outputs were fed back into the control loop for accurate trajectory tracking.

Results

The integrated system achieved stable lane-keeping, smooth trajectory tracking, and successful obstacle avoidance. The PID and LQR controllers maintained low steady-state error, and EKF-SLAM significantly improved pose estimation accuracy in GPS-denied test scenarios. The A* planner consistently generated feasible routes while respecting environmental constraints.
